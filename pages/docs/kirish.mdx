### Saytga bo'limlar qo'shish

Hozirda saytda Server va Golang bo'limi mavjud. Bo'lim ichida mavzular bo'ladi. Siz yangi bo'lim ochmoqchi bo'lsangiz `/page/` jildi ichiga yangi bitta jild oching jild nomi bo'limning nomiga bo'gliq. E'tibor bering jild nomi qisqa va iloji bo'lsa bitta birikmadan iborat bo'lsin.

Jild ochgningizdan keyin jildga kirib `_meta.json` formatidagi fayl ochib oling. Barcha qo'llanmalar `.mdx` fayllariga yoziladi. Jild ichida birorta `.mdx` fayl ochib oling. Masalan `test.mdx`. Endi `_meta.json` fayliga kirib quyidagi kodni kiritishingiz kerak.

```json
{
    "test": "Mavzu sarlavhasi",
    "devops": "DevOps haqida"
  }
```
Bo'limga yangi mavzular qo'shsangiz albatta `_meta.json` ga kiritishingiz kerak vergul qo'ygan holda.
Bu yerda test va devops deb yozilgani `.mdx` fayli nomi kiritiladi keyingisi esa sarlavha. Yangi Bo'lim qo'shganingizda `/pages/` jildi ichidagi `_meta.json` ga xam bo'limingizni kiritishingiz kerak. Masalan Biz test degan bo'lim ochdik.
```json
    "server": "Server",
    "golang": "Golang",
    "test": "Test"
```
Quyidagi bo'limlardan keyin vergul qo'ygan holda o'zingizni bo'limingizni kiritasiz. jild nomi `test  ` edi va salavhaga `Test` deb nom berdik.Bu mavzu qo'shishdek bir xil faqat bo'lim qo'shiladi
